========================================================================
    STATIC LIBRARY : QTrans Project Overview
========================================================================

QTrans - статическая библиотека, объектно-ориентированная обертка над
библиотекой TRANS2QUIK.dll в части подключения и обслуживания транзакций.
Снимает ограничение на загрузку только одного экземпляра библиотеки
TRANS2QUIK, а так же решает проблему делегирования callback-функций
методам объекта.

2013-02-19 Через DDE недоступен идентификатор фирмы. Для получения этой
информации в QTrans добавлен соответствующий функционал.
